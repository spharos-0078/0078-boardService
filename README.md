## 🎯 프로젝트 개요

Board Service는 **다양한 게시판 기능**과 **AI 기반 문의 응답 시스템**을 제공하는 서비스입니다.

### 핵심 기능
- **다중 게시판**: 공지사항, 이벤트, FAQ, 커뮤니티 게시판
- **AI 문의 응답**: OpenAI GPT-4 기반 자동 문의 응답
- **커뮤니티 관리**: 조각 상품별 커뮤니티 게시판
- **콘텐츠 관리**: 게시글 작성, 수정, 삭제, 검색
- **파일 관리**: 이미지 및 파일 업로드 지원

### 게시판 구조
- **공지사항**: 플랫폼 공지 및 업데이트 정보
- **이벤트**: 프로모션 및 이벤트 정보
- **FAQ**: 자주 묻는 질문 및 답변
- **커뮤니티**: 자유 게시판 및 토론
- **상품별 커뮤니티**: 각 조각 상품별 전용 게시판

## 🛠️ 기술 스택

### Backend
- **Java 17** - LTS 버전
- **Spring Boot 3.5.0** - 최신 프레임워크
- **Spring WebFlux** - 반응형 웹 스택
- **Spring Data MongoDB** - NoSQL 데이터베이스
- **Spring Data MongoDB Reactive** - 반응형 MongoDB

### Database & Cache
- **MongoDB 4.4** - 문서형 데이터베이스
- **Redis 7.0** - 캐싱 및 세션 관리

### AI Integration
- **OpenAI GPT-4** - AI 문의 응답 생성
- **OpenAI API** - 자연어 처리 및 응답 생성

### Message Queue
- **Apache Kafka 3.0** - 이벤트 스트리밍
- **Spring Kafka** - Kafka 연동

### Service Discovery
- **Netflix Eureka** - 서비스 등록/발견

### Documentation
- **SpringDoc OpenAPI 3** - API 문서화
- **Swagger UI** - API 테스트

## 🚀 주요 기능

### 1. 게시판 관리
- **게시글 CRUD**: 생성, 조회, 수정, 삭제
- **카테고리 관리**: 게시판 타입별 분류
- **검색 기능**: 제목, 내용, 작성자 검색
- **정렬 기능**: 작성일, 조회수, 추천수 정렬

### 2. 콘텐츠 관리
- **Rich Editor**: 텍스트 에디터 지원
- **파일 업로드**: 이미지 및 파일 첨부
- **태그 관리**: 게시글 태그 기능
- **임시 저장**: 게시글 임시 저장 기능

### 3. 커뮤니티 기능
- **조회수 관리**: 게시글 조회수 증가
- **추천 시스템**: 게시글 추천/비추천
- **신고 기능**: 부적절한 게시글 신고
- **관리자 기능**: 게시글 관리 및 삭제

### 4. AI 문의 응답
- **자동 응답**: GPT-4 기반 문의 자동 응답
- **문맥 이해**: 이전 문의 내역 기반 응답
- **다국어 지원**: 한국어, 영어 문의 응답
- **학습 데이터**: 플랫폼 정보 기반 답변

## 📝 게시판 타입

### 운영 게시판
- **NOTICE**: 공지사항 게시판
- **EVENT**: 이벤트 및 프로모션 게시판
- **FAQ**: 자주 묻는 질문 게시판
- **NEWS**: 뉴스 및 업데이트 게시판

### 커뮤니티 게시판
- **COMMUNITY**: 자유 게시판
- **DISCUSSION**: 토론 게시판
- **REVIEW**: 리뷰 및 후기 게시판
- **PRODUCT_COMMUNITY**: 상품별 커뮤니티

### 고객 지원
- **QNA**: 질문 및 답변 게시판
- **INQUIRY**: 문의 게시판
- **SUPPORT**: 기술 지원 게시판

## ⚙️ 환경 설정

### 필수 요구사항
- Java 17 이상
- MongoDB 4.4 이상
- Redis 7.0 이상
- OpenAI API 키
- Kafka 3.0 이상
- Eureka Server 실행 중

### 환경 변수
- **MONGODB_URI**: MongoDB 연결 정보
- **REDIS_HOST**: Redis 서버 연결 정보
- **OPENAI_API_KEY**: OpenAI API 키
- **KAFKA_HOST**: Kafka 서버 정보
- **EUREKA_HOST**: Eureka 서버 정보
- **FILE_UPLOAD_PATH**: 파일 업로드 경로
- **MAX_FILE_SIZE**: 최대 파일 크기

## 📚 API 문서

### Swagger UI 접속
API 문서는 Swagger UI를 통해 확인할 수 있습니다.

### 주요 API 엔드포인트

#### 게시판 관리
- **게시판 목록 조회**: 특정 타입의 게시판 목록 조회
- **게시글 상세 조회**: 특정 게시글의 상세 정보 조회
- **게시글 작성**: 새로운 게시글 작성
- **게시글 수정**: 기존 게시글 수정
- **게시글 삭제**: 게시글 삭제

#### 검색 및 필터
- **게시글 검색**: 제목, 내용, 작성자로 검색
- **게시글 필터링**: 조건별 게시글 필터링
- **인기 게시글 조회**: 조회수 기준 인기 게시글
- **최신 게시글 조회**: 최신 작성 게시글

#### 파일 관리
- **파일 업로드**: 이미지 및 파일 업로드
- **파일 다운로드**: 첨부 파일 다운로드
- **파일 삭제**: 업로드된 파일 삭제

#### AI 문의 응답
- **AI 문의 요청**: AI 기반 문의 응답 요청
- **AI 응답 조회**: AI 응답 내역 조회

#### 추천 및 신고
- **게시글 추천**: 게시글 추천/비추천
- **게시글 신고**: 부적절한 게시글 신고
- **게시글 통계**: 게시글 조회수, 추천수 통계
